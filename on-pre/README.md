# On-Pre 환경 구성

## 목표

kubeadm 기반 Kubernetes 온프레미스 환경 구축 및 운영

## 주요 작업

- Ubuntu VM 구성
- Kubernetes 클러스터 구축
- Flannel CNI 구성
- ingress-nginx 구성
- Prometheus / Grafana 모니터링 구성

## 디렉토리 설명

- infra/ : VM 및 네트워크 구성
- k8s/ : Kubernetes 리소스 설정
- monitoring/ : Prometheus / Grafana 구성
- docs/ : 온프레미스 관련 문서
