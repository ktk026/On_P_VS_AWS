# AWS 환경 구성

## 목표

AWS EKS 기반 Kubernetes 환경 구축 및 운영

## 주요 작업

- VPC 및 네트워크 구성
- EKS 클러스터 구축
- ECR 구성
- RDS PostgreSQL 구성
- AWS Load Balancer Controller 구성
- CloudWatch 모니터링 구성

## 디렉토리 설명

- infra/ : AWS 인프라 구성
- eks/ : EKS 관련 Kubernetes 설정
- monitoring/ : CloudWatch 및 모니터링 구성
- docs/ : AWS 관련 문서
